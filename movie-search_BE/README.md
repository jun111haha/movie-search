## 주요기능
- 카카오 소셜 로그인 지원
- 외부 API 를 통한 실시간 영화정보 제공
- 내 로그에 영화정보 저장
- 내 로그에 영화정보 삭제
- 내 로그에 TV프로그램 저장
- 내 로그에 TV프로그램 삭제
